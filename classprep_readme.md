###### V3.1.2 ( 게시판 기능 추가 )
클래스프렙 페이지 GNB에 게시판 탭을 추가한다.   
탭 안에는 자료실, 공지사항, 도움말이 추가된다.   
각 컨텐츠별 성격은 다음과 같다.   
1. 자료실 : 클래스프렙이 제안하는 오프라인 강의 컨텐츠
2. 공지사항 : 시스템 및 버전업 공지사항
3. 도움말 : 클래스프렙 도움말
